# Content Researcher Skill

## Purpose
The `content-researcher` skill automates discovering and collating research trends in AI, leadership, and engineering management domains. It combines structured recommendations, automated scripts, and curated resources to aid users' deep analysis and strategy building.

## Scope
This skill enables users to:
1. Extract relevant content from key articles and publications.
2. Analyze external sources for trends and insights in AI, leadership paradigms, and engineering workflows.
3. Assist in constructing content strategies based on compiled data.

## File Structure
- `sources.json`: Contains initial seed research URLs organized by topic.
- `research_trends.js`: JavaScript-based script for initiating new research tasks with APIs or web scrappers.
- `SKILL.md`: This documentation file explaining the purpose, usage, and structure of the `content-researcher` skill.

## Integration
The skill integrates with:
- External APIs (e.g., News APIs for automating research tasks).
- External libraries for AI summarization and data processing.

## Usage
- Populate `sources.json` with preferred URLs for AI, leadership, and engineering management trends.
- Use `research_trends.js` script to perform batch research, generating a digest report of the extracted insights.

This document may evolve with user feedback and research enhancements.
