# Jarvis Clawdbot Configuration

This repository contains the configuration and persona files for the Jarvis Clawdbot assistant.

## Files
- **SOUL.md**: The core personality and behavioral guidelines.
- **USER.md**: Information about the user (Wagner Souza).
- **AGENTS.md**: Workspace and session instructions.
- **IDENTITY.md**: Metadata about the assistant.
- **TOOLS.md**: Environment-specific notes and tool configurations.
- **model_priority.md**: Logic for selecting AI models based on task complexity.
- **whatsapp_autoreply.py**: Script for managing automated WhatsApp responses.

## Purpose
Version control for the assistant's evolution while ensuring no secrets are exposed.
